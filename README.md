# CoffeeNotes

**About**

CoffeeNotes is a webapp for taking notes. Its aim is to be simple, and for my practice in dealing with databases and making a webapp in general.

I'm only listing two features in the below **Features** section for now, as I have not created to many webapps and don't want to get overly ambitious and 
hit a goal before I add more. 

**Features**

**Feature one:** The making and storing of notes.

**Feature two:** Predicted text for faster note taking.
